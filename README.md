# Task Manager Application API
This project implements a Task Management System using React.js, Node.js, Express, and MySQL.

## Setup Instructions

1. **Clone the Repository:**
   git clone https://github.com/ritesh-singh-17/earnest_assignment.git
   cd earnest_assignment

2. **Install Frontend Dependencies in one terminal**
   cd client
   npm install

3. **Install Backend Dependencies in another terminal**
    cd server
    npm install

3. **Set Up Database:**
   Create a MySQL database.
   Import earnest_assignment.sql to your database.

4. **Run the Application:**
   In terminal 1, run command --> npm start
   In terminal 2, run command --> node app.js
